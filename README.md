# Chimera

Chimera is a planned terminal-native, provider-neutral AI coding-agent platform. It is designed to start as a safe local coding CLI and evolve into a multi-agent system where writer, reviewer, and challenger agents can coordinate on complex software tasks.

The current repository contains the initial architecture blueprint:

- [Chimera Terminal Coding Agent Blueprint](docs/chimera-agent-blueprint.md)

The blueprint covers product vision, competitive research, technical architecture, context engineering, prompt engineering, tool permissions, MVP scope, roadmap, OSS stack recommendations, evaluation, open-source collaboration, security, and future multi-agent expansion.
