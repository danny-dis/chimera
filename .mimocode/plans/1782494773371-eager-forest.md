# Plan: Wire LSP Tool to ChimeraLspService

## Context
The `lsp` tool in `chimera-tools` is a stub that throws "not yet implemented". The `ChimeraLspService` in `chimera-lsp` is fully built. We need to connect them.

## Changes

### 1. Add `@chimera/lsp` dependency
**File**: `packages/chimera-tools/package.json`
- Add `"@chimera/lsp": "workspace:*"` to dependencies

### 2. Rewrite LSP tool
**File**: `packages/chimera-tools/src/tools/lsp.ts`

Replace the stub with a real implementation:
- Import `ChimeraLspService` and types from `@chimera/lsp`
- Use a module-level `Map<string, ChimeraLspService>` cache keyed by workspace root
- On each tool call, get or create the service for the workspace
- Dispatch to the appropriate method based on `params.operation`
- Format results as human-readable strings

Operations to implement:
- `goToDefinition` → `service.goToDefinition(filePath, line, character)`
- `findReferences` → `service.findReferences(filePath, line, character)`
- `hover` → `service.hover(filePath, line, character)`
- `documentSymbol` → `service.documentSymbols(filePath)`
- `workspaceSymbol` → `service.workspaceSymbols(query)`

### 3. Export LSP tool in `allTools`
**File**: `packages/chimera-tools/src/index.ts`
- Add `lspTool` to the `allTools` array
