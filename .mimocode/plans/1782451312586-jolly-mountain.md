# One-Line Install Command

## Goal
Enable `npx @chimera/cli` (or `bunx`) + a `curl | bash` fallback installer.

## Changes

### 1. Fix `packages/chimera-cli/package.json` for npm publish
- Add `"files": ["dist", "README.md"]` — only ship built output
- Add `"license": "MIT"`, `"repository"`, `"engines": { "node": ">=20" }`
- Add `"publishConfig": { "access": "public" }`

### 2. Create `install.sh` at repo root
- Checks for Node >= 20
- `git clone` + `pnpm install` + `pnpm build`
- Prints success message with usage instructions

### 3. Update `README.md`
- Replace "Getting Started" with one-liner install section
- Show both `npx` and `curl` options

### 4. Publish workflow
- Script: `pnpm publish --filter @chimera/cli` (pnpm rewrites `workspace:*` to actual versions)

## Files
- `packages/chimera-cli/package.json` — add files, license, engines, publishConfig
- `install.sh` — new file
- `README.md` — update Getting Started section

## Verification
1. `cd packages/chimera-cli && npm pack --dry-run` — verify only dist/ is included
2. `bash install.sh` — test the curl installer locally
3. `npx @chimera/cli --help` — test after publish
