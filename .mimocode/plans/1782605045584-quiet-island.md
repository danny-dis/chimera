## Phase 2: Version Bump to 0.1.5

### 2a. Create `scripts/sync-versions.sh`

A script that sets the root `package.json` version and syncs it to all workspace packages.

**File**: `scripts/sync-versions.sh`
```bash
#!/bin/bash
# Sync version across all workspace packages
VERSION=${1:?Usage: sync-versions.sh <version>}
# Update root
jq --arg v "$VERSION" '.version = $v' package.json > tmp && mv tmp package.json
# Update all packages
for pkg in packages/chimera-*/package.json; do
  jq --arg v "$VERSION" '.version = $v' "$pkg" > tmp && mv tmp "$pkg"
done
echo "✓ All packages set to $VERSION"
```

## Phase 3: Build Release Pipeline

### 3a. Enhanced CI workflow

**File**: `.github/workflows/ci.yml` — already exists, keep as-is (build/lint/typecheck/test on push/PR).

### 3b. New Release workflow

**File**: `.github/workflows/release.yml`

Triggers on `v*` tags. Steps:
1. Checkout
2. Setup pnpm + Node 20
3. Install dependencies
4. Build all packages
5. Run full test suite
6. Publish `@chimera/cli` to npm (using `NPM_TOKEN` secret)
7. Create GitHub Release with changelog

```yaml
name: Release
on:
  push:
    tags: ['v*']

jobs:
  release:
    runs-on: ubuntu-latest
    permissions:
      contents: write
      id-token: write
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v4
        with: { version: 11 }
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: pnpm
          registry-url: 'https://registry.npmjs.org'
      - run: pnpm install --frozen-lockfile
      - run: pnpm build
      - run: pnpm lint
      - run: pnpm typecheck
      - run: pnpm test
      - name: Publish CLI
        run: pnpm --filter @chimera/cli publish --no-git-checks
        env:
          NODE_AUTH_TOKEN: ${{ secrets.NPM_TOKEN }}
      - name: Create GitHub Release
        uses: softprops/action-gh-release@v2
        with:
          generate_release_notes: true
```

### 3c. E2E smoke test workflow

**File**: `.github/workflows/e2e-smoke.yml`

Triggers on push to `main` (with secrets). Tests all modes/presets with real API keys.

```yaml
name: E2E Smoke Tests
on:
  push:
    branches: [main]

jobs:
  smoke:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        mode: [ask, plan, code, debug, review, oal]
        preset: [solo, duo, trio, fusion]
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v4
        with: { version: 11 }
      - uses: actions/setup-node@v4
        with: { node-version: 20, cache: pnpm }
      - run: pnpm install --frozen-lockfile
      - run: pnpm build
      - name: Smoke test ${{ matrix.mode }}/${{ matrix.preset }}
        run: node scripts/smoke-test.js --mode ${{ matrix.mode }} --preset ${{ matrix.preset }}
        env:
          GOOGLE_API_KEY: ${{ secrets.GOOGLE_API_KEY }}
          CHIMERA_CHEAP_API_KEY: ${{ secrets.CHIMERA_CHEAP_API_KEY }}
          CHIMERA_CHEAP_BASE_URL: ${{ secrets.CHIMERA_CHEAP_BASE_URL }}
          CHIMERA_CHEAP_MODEL: ${{ secrets.CHIMERA_CHEAP_MODEL }}
```
