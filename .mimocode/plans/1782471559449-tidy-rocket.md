# Plan: Redesign Duo to Writerâ†’Reviewer Loop

## Goal

Change duo from "writer drafts â†’ reviewer overwrites" to a loop:
1. Writer drafts
2. Reviewer verifies â†’ returns **pass** / **fail** / **improve** with feedback
3. If **fail** or **improve**: feedback goes back to writer, writer revises
4. Loop until reviewer says **pass**, or max revisions reached
5. Final output = writer's last-passed draft

### 3. `duo-executor.ts` â€” New `executeWithAnalysis` flow

```
1. Recursion guard + config validation (unchanged)
2. Stage 1: Writer produces initial draft
3. Budget check
4. Optional linter on draft
5. Loop (maxRevisions iterations):
   a. Reviewer evaluates draft â†’ verdict + feedback
   b. Budget check
   c. If verdict === 'pass' â†’ break (done)
   d. If verdict !== 'pass' and revisions remaining:
      - Writer revises with feedback
      - Budget check
      - Increment revision counter
      - Continue loop
   e. If no revisions left â†’ break (degraded)
6. Synthesize: use the writer's last draft as final output
7. Sources: track all writer/reviewer calls with revision numbers
```

**Key behaviors:**
- `pass` â†’ loop ends immediately, writer's draft is the final output
- `fail`/`improve` + revisions left â†’ writer revises, loop continues
- `fail`/`improve` + no revisions left â†’ writer's last draft is output, `degraded: true` with reason "max revisions reached"
- Budget exceeded at any point â†’ return last available draft, `degraded: true`
- Reviewer parse failure (no recognizable verdict) â†’ treat as `improve` with the raw reviewer text as feedback

### 4. Verdict parsing

The reviewer returns free-form text. We parse the first line for `VERDICT: pass|fail|improve` (case-insensitive, flexible whitespace). If no verdict found, fallback to `improve` with raw text as feedback.

```typescript
private parseVerdict(raw: string): { verdict: ReviewVerdict; feedback: string } {
  const firstLine = raw.split('\n')[0];
  const match = firstLine.match(/VERDICT:\s*(pass|fail|improve)/i);
  const verdict: ReviewVerdict = match ? (match[1].toLowerCase() as ReviewVerdict) : 'improve';
  const feedback = match ? raw.slice(raw.indexOf(match[0]) + match[0].length).trim() : raw;
  return { verdict, feedback };
}
```
